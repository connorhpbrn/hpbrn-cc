# hpbrn.co

My personal site — who I am, what I'm building, and how to reach me.

UK-based solo builder making AI-native tools: the infrastructure that gives
agents real context and tool access. Open-source, and clean enough to be
judged in the real world.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- [Geist](https://vercel.com/font)

## Develop

```bash
npm install
npm run dev
```

Then open [localhost:3000](http://localhost:3000).
